Project Overview

The grocery app is a mobile application designed to allow users to generate personalized meal plans and grocery lists from both structured profile data and free-text natural language input. The application simulates intelligent behavior through keyword matching, contextual analysis, and deduplication of grocery items, while also enforcing user constraints such as allergens, dietary preferences, and budget.
